## Usage

You can change default API in client/src/actions/api.js file:

    const baseUrl = "http://localhost:5000/api/";


### Install Dependencies & Run

```
cd client
npm install

# Run Frontend + Backend
npm run dev

# Run Backend only
npm run server
```
